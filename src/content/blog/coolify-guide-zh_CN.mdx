## 10 分钟部署 Coolify

### 前置要求

- 一台 Linux 服务器（Ubuntu 22.04+ 推荐）
- 2GB+ 内存（4GB 更好）
- Docker 和 Docker Compose（Coolify 会自动检查和安装）
- 一个域名（可选但推荐）

### Step 1：一键安装

```bash
curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash
```

就这一条命令。Coolify 会自动：
- 检查 Docker 是否安装（没有就帮你装）
- 拉取所有必要的镜像
- 启动 Coolify 服务
- 配置反向代理

安装完你会看到类似这样的输出：

```
Coolify v4.0.0-beta.xxx
- Configuration: /data/coolify
- Port: 8000
- Database: PostgreSQL 15
```

**踩坑提醒：** 如果你是国内服务器，这一步可能比较慢。两种解决方式：一是配 Docker 镜像加速器，二是用海外 VPS 先装好再迁回来。

### Step 2：访问管理面板

浏览器打开 `http://你的服务器IP:8000`

首次访问会让你：
1. 创建管理员账号
2. 输入你的服务器 IP 或域名
3. 选择安装路径（默认 `/data/coolify`）

**注意：** 如果 8000 端口访问不了，检查防火墙：

```bash
# Ubuntu UFW
sudo ufw allow 8000/tcp
sudo ufw reload

# 或者如果你用的是云服务器，去安全组放行 8000
```

### Step 3：配置域名和 SSL（推荐）

在管理面板 → Settings 中：
1. 输入你的域名
2. Coolify 会自动配置 Let's Encrypt SSL 证书
3. 之后你访问管理面板就是 HTTPS 了

### Step 4：连接 GitHub/GitLab

1. 进入 Settings → GitHub / GitLab
2. 点击「Login with GitHub/GitLab」
3. 授权 Coolify 访问你的仓库

授权完成后，Coolify 能直接拉取你的代码并自动部署。

### Step 5：部署第一个项目

1. 点击「New Application」
2. 选择「Git Based」
3. 选择你的仓库
4. Coolify 会自动检测框架（Next.js、Nuxt、Django、Go...）
5. 点击「Deploy」

等 1-2 分钟，你的项目就上线了。访问 `https://yourapp.com` 看看。

## 生产环境踩过的坑

跑了 8 个月，总结几个真实的坑：

### 1. 内存不够别硬上

Coolify 本身 + 你部署的应用 + 数据库，至少需要 4GB 内存。2GB 能跑但会很卡。

**建议：** 至少 4GB，推荐 8GB。预算够的话上 16GB。

### 2. 备份一定要配

Coolify 的数据存在 `/data/coolify` 目录。你必须自己配备份。

```bash
# 简单的备份脚本示例
#!/bin/bash
DATE=$(date +%Y%m%d)
tar -czf /backup/coolify_$DATE.tar.gz /data/coolify
# 保留最近 7 天
find /backup -name "coolify_*.tar.gz" -mtime +7 -delete
```

加到 crontab：

```bash
0 3 * * * /opt/backup/coolify-backup.sh
```

### 3. 更新前先看 Changelog

Coolify 还在 beta 阶段，偶尔会有 breaking changes。更新前先看 GitHub 的 release notes。

```bash
# 更新 Coolify
curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash
```

### 4. 域名 DNS 别忘了

很多人部署完了发现域名访问不了——90% 的情况是 DNS 没配对。确保你的域名 A 记录指向服务器 IP。

## 常见问题

### Q: Coolify 稳定吗？生产环境能用吗？

**A:** 我的 4 个项目在上面跑了 8 个月，没出过一次事故。社区活跃，bug 修复很快。适合中小项目和个人项目。

### Q: 和 Dokku 有什么区别？

**A:** Dokku 是 Heroku 的命令行替代，Coolify 是 Heroku + Vercel 的图形化替代。Coolify 有 Web UI、自动 SSL、多服务器管理，体验好很多。

### Q: 支持什么语言/框架？

**A:** 几乎所有。Coolify 通过 Nixpacks 自动检测：Next.js、Nuxt、Django、Flask、Go、Rust、Ruby on Rails、PHP/Laravel... 不能自动检测的也可以手动配置 Dockerfile。

### Q: 服务器在国外还是国内？

**A:** 都行。国内服务器需要注意：1) 安装时可能需要 Docker 镜像加速；2) 访问 GitHub 仓库可能需要代理。推荐海外 VPS（Vultr、DigitalOcean）。

### Q: 要花多少钱？

**A:** Coolify 本身完全免费开源。你只需要一台服务器的钱。$5/月的 VPS 就够跑好几个小项目了。
